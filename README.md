ENGLISH/
This is an AI project I developed using a CNN model pre-trained with the Oxford pet dataset, and it features a user interface. You can find the project code and a general report describing the project in the files. This was my first AI project where I worked with PyQt interface programming. I completed it in June 2022, during the second semester of my third year of university. You can use it however you like. Although I'm no longer actively working on AI development, you can contact me as you wish.
